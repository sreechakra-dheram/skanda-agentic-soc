PROJECT TITLE

SkANDA Prime — Agentic AI Security Operations Center (SOC)

PROJECT OVERVIEW

SkANDA Prime is an AI-powered autonomous Security Operations Center (SOC) designed to monitor a laptop’s real network traffic, detect cyber threats using an LLM (Gemini API), reason about attacks in real time, and optionally execute automated mitigation actions.

The system uses:

Real network logs (Zeek)

Gemini API (single API key)

Node.js backend

WebSocket real-time streaming

Agentic architecture (logical multi-agent using one model)

Existing advanced SOC dashboard UI (already built)

The UI must remain unchanged stylistically.

CURRENT SYSTEM GOAL

Transform an animated SOC dashboard into a real AI cybersecurity monitoring system capable of:

Monitoring laptop network traffic

Detecting anomalies using AI reasoning

Streaming AI thinking live

Generating forensic reports

Performing autonomous mitigation (with safety controls)
